Crear un pipeline CI/CD en AWS de tres etapas (Source, Build y Deploy) que automatice la actualización de una función Lambda existente. El pipeline debe obtener el código desde un repositorio de GitHub cuando se realice un push al branch principal, empaquetar los archivos con CodeBuild y desplegarlos automáticamente en la Lambda mediante CodePipeline. La función destino debe llamarse hello-world-demo y, al finalizar, el pipeline debe actualizarla correctamente con el nuevo código. Se evaluará que las tres etapas funcionen sin errores, que los permisos IAM estén correctamente configurados, que los logs se generen en CloudWatch y que el pipeline ejecute todo el flujo de forma automática al detectar cambios en el código fuente.
